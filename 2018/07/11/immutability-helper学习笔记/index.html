immutability-helper 学习笔记

Posted on 2018-07-11 | In Technology , FontEnd

immutability-helper

不改变原始数据生成该数据的拷贝

安装

可接受命令

  • {$push: array} 对目标 push() array内所有项。
  • {$unshift: array} 对目标 unshift() array内所有项。
  • {$splice: array of arrays} 对目标依据array内每一项执行 splice()。⚠️注意：array内的每一项会被依次的使用，所以array子项的顺序很重要。目标索引在操作过程中可能会变化。
  • {$set: any} 对目标进行全部替换。
  • {$toggle: array of strings} 对目标对象中的一些布尔值进行取反操作。
  • {$unset: array of strings} 对移除目标对象移除某些key。
  • {$merge: object} 对目标对象的某个key对应的value与object进行合并。
  • {$apply: function} 当前value作为function的入参，其函数返回值将更新为新的value。
  • {$add: array of objects} 对 Map 和 Set 增加一个value。当目标是 Set 时，传入的是一个对象数组；当目标是 Map 时，传入的是一个子项为 [key, value] 格式的数组，如：update(myMap, {$add: [['foo', 'bar'], ['baz', 'boo']]})。
  • {$remove: array of strings} 从一个 Map 或 Set 中移除某些key。
