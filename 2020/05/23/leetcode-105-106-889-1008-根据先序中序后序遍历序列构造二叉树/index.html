leetcode 105.从前序与中序遍历序列构造二叉树 [Medium]

题目地址

https://leetcode-cn.com/problems/construct-binary-tree-from-preorder-and-inorder-traversal/

题目描述

根据一棵树的前序遍历与中序遍历构造二叉树。

注意:
你可以假设树中没有重复的元素。

例如，给出

前序遍历 preorder = [3,9,20,15,7]
中序遍历 inorder = [9,3,15,20,7]

返回如下的二叉树：

   3
   /   \
   9  20
  /  \
 15  7

题解

思路：递归构造子树
前序遍历： [ 根,  [左子树的前序遍历],  [右子树的前序遍历] ]
中序遍历： [ [左子树的中序遍历],  根,  [右子树的中序遍历] ]

leetcode 105.从中序与后序遍历序列构造二叉树 [Medium]

题目描述

根据一棵树的中序遍历与后序遍历构造二叉树。

注意:
你可以假设树中没有重复的元素。

例如，给出

后序遍历 postorder = [9,15,7,20,3]

   15  7

中序遍历： [ [左子树的中序遍历],  根,  [右子树的中序遍历] ]
后序遍历： [ [左子树的后序遍历],  [右子树的后序遍历],  根 ]

leetcode 889.根据前序和后序遍历构造二叉树 [Medium]

题目地址

https://leetcode-cn.com/problems/construct-binary-tree-from-preorder-and-postorder-traversal/

题目描述

返回与给定的前序和后序遍历匹配的任何二叉树。

pre 和 post 遍历中的值是不同的正整数。

示例：

输入：pre = [1,2,4,5,3,6,7], post = [4,5,2,6,7,3,1]
输出：[1,2,3,4,5,6,7]

提示：

  • 1 <= pre.length == post.length <= 30
  • pre[] 和 post[] 都是 1, 2, …, pre.length 的排列
  • 每个输入保证至少有一个答案。如果有多个答案，可以返回其中一个。

题解

思路：递归构造子树
前序遍历： [ 根,  [左子树的前序遍历],  [右子树的前序遍历] ]
后序遍历： [ [左子树的后序遍历],  [右子树的后序遍历],  根 ]

前序遍历的第一项以及后序遍历的最后一项一定是当前树的根，重点是如何把另外的序列区分出左子树和右子树的遍历序列。利用子树的前序遍历和后序遍历的根依旧是一个根在头、一个根在尾，所以从头遍历后序序列，找到和前序序列第 1 项（第 0 项是左右子树的父亲节点）相同的位置，则可以得到左子树后序序列的长度，则可以拆分左右子树遍历序列，进入下一层递归。

leetcode 1008.先序遍历构造二叉搜索树 [Medium]

题目地址

https://leetcode-cn.com/problems/construct-binary-search-tree-from-preorder-traversal/

题目描述

返回与给定先序遍历 preorder 相匹配的二叉搜索树（binary search tree）的根结点。

(回想一下，二叉搜索树是二叉树的一种，其每个节点都满足以下规则，对于 node.left 的任何后代，值总 < node.val，而 node.right 的任何后代，值总 > node.val。此外，先序遍历首先显示节点的值，然后遍历 node.left，接着遍历 node.right。）

示例：

输入：[8,5,1,7,10,12]
输出：[8,5,10,1,7,null,12]

   8
   /   \
   5  10
  /  \   \
  1  7  12

提示：

  • 1 <= preorder.length <= 100
  • 先序 preorder 中的值是不同的。

题解

解法一

思路：利用二叉搜索树中序遍历一定是升序这一大特点，将先序序列排序得到中序序列，然后转化为上面👆105 题。

解法二

思路：不使用递归而使用迭代

  • 首先将第一个数据入栈
  • 然后遍历其余节点与当前节点（栈顶）比较 
    - 如果小于当前节点，直接放在左边  
- 否则从栈顶向下寻找第一个比当前节点小的节点，放到该节点的右边
  • 将当前节点入栈，继续处理后续节点
